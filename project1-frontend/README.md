# Project 1 Frontend - SciSciNet UMD Visualization

React + D3.js visualization for UMD Computer Science research data.

## Setup

```bash
cd project1-frontend
npm install
npm start
```

App runs at http://localhost:3000

**Note**: Start the backend first at http://localhost:5000

## Features

### T1: Networks
- Citation Network (force-directed, draggable nodes)
- Collaboration Network (author connections)

### T2: Timeline & Patents
- Publication timeline (past 10 years)
- Patent citation histogram
- Interactive filtering by year

### T3: Optimized Network
- Radial layout
- Hierarchical edge bundling
- Adjustable bundling strength

## Tech Stack

- React 18 + TypeScript
- D3.js v7

## Available Scripts

- `npm start` - Run development server
- `npm run build` - Build for production
- `npm test` - Run tests

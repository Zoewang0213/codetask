# Project 2 Frontend - SciSciNet LLM Agent Chat Interface

React chat interface with Vega-Lite visualization for UMD CS research data analysis.

## Setup

```bash
cd project2-frontend
npm install
npm start
```

App runs at http://localhost:3000

**Note**: Start the backend first at http://localhost:5001

## Features

- Chat interface powered by Claude AI
- Automatic Vega-Lite chart generation
- Quick question buttons
- Real-time message streaming

## Tech Stack

- React 18 + TypeScript
- Vega-Lite for data visualization
- Anthropic Claude API (via backend)

## Example Questions

- "How many papers were published each year from 2020 to 2024?"
- "Who are the top 10 authors by paper count?"
- "What are the citation statistics?"
- "Show me the yearly trend in patent citations"

## Available Scripts

- `npm start` - Run development server
- `npm run build` - Build for production
- `npm test` - Run tests
